# 손에 잡히는 10분 정규표현식

## 0. 시작하기 전에

- 정규 표현식은 왜 사용하는가?
  - 일치하는 텍스트 찾기 (검색)
  - 텍스트를 찾은 뒤에 그것을 원하는 텍스트로 바꾸기 (치환)
- 정규표현식을 사용하면 문제를 해결하는 방법이 하나만은 아님을 알게 된다. 정규표현식을 사용하는 데 올바르거나 틀린 방법은 없다. (제대로 동작하기만 한다면)
  - 어떤 방식은 더 간단하고
  - 어떤 방식은 더 빠르며
  - 어떤 방식은 다른 환경에 이식하기 쉽고
  - 어떤 방식은 더 많은 일을 수행할 수 있다
- 다른 프로그래밍 언어와 마찬가지로 연습하고, 연습하고, 또 연습하라. 책에 나오는 예제들을 모두 직접 실행해 보라.

## 1. 기본적인 검색 문자 및 규칙

자세한 내용은 [MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_Expressions) 문서 참고.

### 1.1 기본 메타문자

| 문자  |               규칙               |
| :---: | :------------------------------: |
|   .   |         모든 문자와 일치         |
|  \|   |     왼쪽 혹은 오른쪽과 일치      |
| \[ \] | 문자 집합 구성원 중 하나와 일치  |
| \[^\] | 문자 집합 구성원을 제외하고 일치 |
|   -   |   범위 정의([A-Z]와 같은 형태)   |
|  \    |  다음에 오는 문자를 이스케이프   |

### 1.2 수량자

| 문자  |                       규칙                       |
| :---: | :----------------------------------------------: |
|  \*   | 문자가 없는 경우 or 하나 이상 연속하는 문자 찾기 |
|  \*?  |                  게으른 \* 문자                  |
|   +   |               문자 하나 이상 찾기                |
|  +?   |                  게으른 + 문자                   |
|   ?   |          문자가 없거나 하나인 문자 찾기          |
|  {n}  |              정확히 요소와 n번 일치              |
| {m,n} |              요소와 m에서 n번 일치               |
| {n,}  |               요소와 n번 이상 일치               |
| {n,}? |                   게으른 {n,}                    |

### 1.3 위치 지정

| 문자 |         규칙         |
| :--: | :------------------: |
|  ^   | 문자열의 시작과 일치 |
|  \A  | 문자열의 시작과 일치 |
|  $   |  문자열의 끝과 일치  |
|  \Z  |  문자열의 끝과 일치  |
|  \<  |  단어의 시작과 일치  |
| \\>  |   단어의 끝과 일치   |
|  \b  |   단어 경계와 일치   |
|  \B  |   \b와 반대로 일치   |

### 1.4 특수한 문자

|  문자  |           규칙            |
| :----: | :-----------------------: |
| \[\b\] |        백스페이스         |
|   \c   |     제어 문자와 일치      |
|   \d   |     모든 숫자와 일치      |
|   \D   |         \d와 반대         |
|   \f   |  페이지 넘김 (form feed)  |
|   \n   |          줄바꿈           |
|   \r   |        캐리지 리턴        |
|   \s   |     공백 분자와 일치      |
|   \S   |     \s와 반대로 일치      |
|   \t   |            탭             |
|   \v   |          수직 탭          |
|   \w   | 영숫자 문자나 밑줄과 일치 |
|   \W   |     \w와 반대로 일치      |
|   \x   |     16진수 값과 일치      |
|   \0   |      8진수 값과 일치      |

### 1.5 역참조와 전후방탐색

|            문자             |                                    규칙                                     |
| :-------------------------: | :-------------------------------------------------------------------------: |
|             ()              |                              하위 표현식 정의                               |
|             \1              | 첫 번째 일치한 하위 표현식. 두 번째 일치한 하위 표현식은 \2로 표기하는 방식 |
|             ?=              |                                  전방탐색                                   |
|             ?<=             |                                  후방탐색                                   |
|             ?!              |                               부정형 전방탐색                               |
|             ?<!             |                               부정형 후방탐색                               |
|    ?(backreference)true     |                                  조건 지정                                  |
| ?(backreference)true\|false |                            else 표현식 조건 지정                            |

### 1.6 대소문자 변환

| 문자 |                    규칙                    |
| :--: | :----------------------------------------: |
|  \E  |           \L 혹은 \U 변환을 끝냄           |
|  \l  |      다음에 오는 글자를 소문자로 변환      |
|  \L  | \E를 만날 때까지 모든 문자를 소문자로 변환 |
|  \u  |      다음에 오는 글자를 대문자로 변환      |
|  \U  | \E를 만날 때까지 모든 문자를 대문자로 변환 |

### 1.7 변경자

| 문자 |    규칙     |
| :--: | :---------: |
| (?m) | 다중행 모드 |
