# 문자 하나 찾기

## 1. 문자 그대로 찾기

예문

> Hello, my name is Ben.

정규표현식

```
Ben
```

결과

> Hello, my name is **Ben**.

### 얼마나 많이 일치하는가?

- 대다수 정규 표현식 구현에서는 일치하는 목록을 모두 얻을 수 있다. (대개 배열 형태로 반환)
  - 예) 자바스크립트에서는 g(global) 플래그를 사용하여 일치된 결과물을 모두 담은 배열을 반환

### 대소문자 다루기

- 대다수 정규 표현식 구현에서는 대소문자 구별을 무시하는 기능이 있다.
  - 기본적으로는 대소문자를 구별하기 때문에 Ben은 ben과는 일치하지 않음
  - 예) 자바스크립트에서는 i 플래그를 사용하여 대소문자 구별을 무시하고 검색

## 2. 모든 문자 찾기

예문

```
sales1.xls
orders3.xls
sales2.xls
sales3.xls
apac1.xls
europe2.xls
na1.xls
na2.xls
sa1.xls
```

정규표현식

```
sales.
```

결과

> **sales1**.xls

> **sales2**.xls

> **sales3**.xls

### 마침표(.)의 역할

- 어떠한 문자나 알파벳, 숫자, 문장 부호로 쓰인 마침표와도 일치
- 여러 개를 동시에 사용할 수도 있음

정규표현식

```
.a.
```

결과

> **sal**es1.xls

> **sal**es2.xls

> **sal**es3.xls

> a**pac**1.xls

> **na1**.xls

> **na2**.xls

> **sa1**.xls

- 만약 `.a.` 뒤에 바로 마침표가 따라오는 문자열과 일치시키려면?
  - 정규 표현식에서 특별한 의미로 쓰이는 마침표가 아니라 진짜 마침표를 찾고 싶다고 알려줘야 함.
  - 이럴 때 역슬래시(\\: 메타문자)를 붙임
  - 따라서 그냥 `.` 는 모든 문자와 일치하지만 `\\.` 는 마침표와 일치

정규표현식

```
.a.\.
```

결과

> **na1.**xls

> **na2.**xls

> **sa1.**xls
